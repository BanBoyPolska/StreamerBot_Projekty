🎯 Minimalistyczny Sub Goal

Dynamiczny pasek postępu subskrypcji dla OBS z panelem kontrolnym i pełną integracją Streamer.bot. Minimalistyczny wygląd, w pełni konfigurowalny.

OBS Studio Streamer.bot Browser Source Twitch HTML/CSS/JS C#
📁 Pliki projektu
PlikRolaGdzie używać
subgoal-bar.html Pasek postępu widoczny na streamie OBS → Browser Source
control-panel.html Panel sterowania, ustawienia, testy Przeglądarka
SubGoal Subscribe.cs Obsługuje nowe i renew suby Streamer.bot
SubGoal Gift.cs Obsługuje gifted subs (1 lub wiele) Streamer.bot
SubGoal Sync.cs Synchronizuje licznik z Twitch API Streamer.bot
SubGoal Send.cs Relay eventów z panelu do OBS Streamer.bot
📦 Instalacja
1
Pobierz pliki
Kliknij Pobierz pliki i umieść wszystkie pliki w jednym folderze, np.: 
C:\subgoal\
  subgoal-bar.html
  control-panel.html
2
Zaimportuj akcje do Streamer.bot
Skopiuj poniższy kod importu, a następnie w Streamer.bot przejdź do Actions → Import, wklej kod i kliknij Import. Zostaną automatycznie utworzone 4 akcje: SubGoal Subscribe, SubGoal Gift, SubGoal Sync, SubGoal Send.

3
Ustaw triggery w Streamer.bot
Przypisz triggery do akcji:
• SubGoal Subscribe → Twitch → Subscribe + Twitch → Re-Subscribe
• SubGoal Gift → Twitch → Gift Subscription
• SubGoal Sync — wywoływana automatycznie przez inne akcje
4
Dodaj pasek do OBS
W OBS dodaj nowe źródło Browser Source o nazwie Sub Goal. W polu URL wpisz lokalną ścieżkę do pliku: 
file:///C:/subgoal/subgoal-bar.html
🔑 Ważne: W ustawieniach źródła zaznacz opcję „Allow OBS to control audio" oraz „Allow OBS WebSocket to control browser source" — bez tego pasek nie będzie reagować na suby.
5
Uruchom panel kontrolny
Otwórz control-panel.html w przeglądarce. Panel pozwala konfigurować wygląd, cel, testować animacje i synchronizować licznik z Twitch API. Połączy się automatycznie ze Streamer.bot przez WebSocket na porcie 8080.
🔄 Konfiguracja SubGoal Sync

Akcja SubGoal Sync wymaga dwóch sub-actions w odpowiedniej kolejności:

1
Twitch → Subscribers → Get Subscriber Count
Pobiera aktualną liczbę subów z Twitch API i zapisuje ją w argumencie subscriberCount.
2
Execute C# Code → SubGoal Sync
Odczytuje subscriberCount i wysyła update do paska OBS oraz panelu przez WebSocket.
🔧 Rozwiązywanie problemów
ProblemPrzyczynaRozwiązanie
Pasek nie reaguje na suby Brak opcji „Allow OBS control" w Browser Source Zaznacz tę opcję w ustawieniach źródła OBS
Panel pokazuje 0 subów SubGoal Sync nie ma sub-action Get Subscriber Count Dodaj go jako pierwszy krok przed Execute Code
Panel nie łączy się ze Streamer.bot WebSocket Server wyłączony lub zły port Włącz WebSocket Server na porcie 8080 w Streamer.bot
Błąd kompilacji CS1061 Nieistniejąca metoda CPH.* Użyj dokładnie kodów z pobranych plików
RunAction nie działa Zła nazwa akcji (podkreślnik vs spacja) Sprawdź czy nazwy akcji mają spacje, nie podkreślniki
