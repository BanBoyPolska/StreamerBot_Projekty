📺 LG WebOS Connector

Plugin do Elgato Stream Deck umożliwiający sterowanie telewizorami LG WebOS przez sieć lokalną — bez pilota, bez dodatkowych aplikacji.

Stream Deck LG WebOS LAN C# BanBoyPolska
⚙️ Wymagania
📺
LG TV z WebOS 4.0+
Modele od 2018 roku wzwyż
🌐
Ta sama sieć LAN/Wi-Fi
TV i komputer muszą być w tej samej sieci
🎛️
Stream Deck Software 6.0+
Dowolny model Stream Deck
🔓
Włączone urządzenia zewnętrzne
Ustawienia → Ogólne → Urządzenia zewnętrzne → Włączone
📦 Instalacja
1
Pobierz plugin
Kliknij przycisk Pobierz plugin po prawej stronie, aby pobrać plik com.banboy.lgtv-webos-connect.streamDeckPlugin.
2
Zainstaluj w Stream Deck
Kliknij dwukrotnie na pobrany plik. Aplikacja Stream Deck automatycznie go zainstaluje i uruchomi — nie potrzebujesz robić nic więcej.
3
Sparuj telewizor
Przeciągnij przycisk Pair TV na wybrany klawisz Stream Deck. W panelu ustawień wpisz adres IP telewizora i kliknij Paruj TV. Na ekranie TV pojawi się dialog potwierdzający — zatwierdź go.
💡 Parowanie wystarczy zrobić raz. Wygenerowany klucz jest zapisywany globalnie — wszystkie przyciski działają automatycznie bez ponownego parowania.
4
Dodaj akcje
Przeciągnij dowolne akcje z listy LGTV WebOS na wybrane przyciski Stream Deck. Gotowe!
🎮 Dostępne akcje
🔌
Switch HDMI Input
Przełącza wejście telewizora na wybrany port HDMI (1, 2, 3 lub 4)
🔇
Toggle Mute
Przełącza wyciszenie dźwięku telewizora (toggle — włącz/wyłącz)
⏻
Power Off
Wyłącza telewizor (tryb standby)
📡
Pair TV
Jednorazowe parowanie z telewizorem. Wymaga podania adresu IP TV.
📍 Jak znaleźć IP telewizora? Na TV: Ustawienia → Ogólne → Sieć → Informacje o połączeniu sieciowym → Adres IP
🔧 Rozwiązywanie problemów
❌ Nie mogę połączyć się z TV
TV jest wyłączony lub w trybie standby. Włącz telewizor przed próbą połączenia.
🌐 Connection refused / timeout
Upewnij się, że TV i komputer są w tej samej sieci. Włącz zdalny dostęp w ustawieniach TV: Ustawienia → Ogólne → Urządzenia zewnętrzne → Włączone.
🔑 Przyciski przestały działać po czasie
Zrestaruj aplikację Stream Deck. Jeśli problem się powtarza — sparuj ponownie używając przycisku Pair TV z aktualnym IP telewizora.
